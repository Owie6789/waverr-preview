## ✨ The Waverr Experience

Waverr 2.0 is a high-performance, offline-first Progressive Web App (PWA) designed for audiophiles who value privacy and design. By bridging the gap between local storage and high-end audio engineering, Waverr transforms your browser into a professional-grade workstation.

### Key Highlights
* **🎧 Pro Audio Engine**: 10-band parametric equalizer & dynamic range compressor.
* **🔍 Instant Search**: Powered by **FlexSearch** for sub-millisecond library indexing.
* **🔮 Midnight Glass UI**: Premium aesthetic with heavy backdrop blurs (40px) and GPU-accelerated transitions.
* **💾 Local First**: Zero tracking. Zero cloud. Your music stays in your browser via **IndexedDB**.

## 📐 Design Philosophy

Waverr follows an **"Application-First"** approach rather than a "Web-First" one:
* **🖱 Micro-interactions**: Every click triggers a haptic visual scale-down effect.
* **📏 Smart Layouts**: 24px corner radii and standardized 8px spacing for a cohesive visual rhythm.
* **🔡 Typography**: Bold headers for branding balanced with Monospace for technical audio data.

## 🚀 Getting Started

### 1️⃣ Installation

**Option A: For Developers (Git)**
```bash
git clone https://github.com/Owie6789/waverr-preview.git
```

**Option B: Manual Download (ZIP)**
* Click the green **"Code"** button at the top of this page.
* Select **"Download ZIP"**.
* Extract the folder to your preferred location.

### 2️⃣ Launch
* **🌐 Browser**: Simply open `index.html` in any modern browser.
* **💡 Pro Tip**: Chrome, Edge, or Brave are recommended for optimal Web Audio API performance.

### 3️⃣ Import Music
* **📂 Library**: Drag and drop your music folder directly into the window. 
* **⚡ Automation**: Waverr will automatically index your files and extract high-quality metadata.
